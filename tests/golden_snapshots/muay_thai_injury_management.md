# FIGHT CAMP PLAN

## PHASE 1: GENERAL PREPARATION PHASE (GPP) - 4 WEEKS (28 DAYS)

### Mindset Focus

- **Confidence:** Track one daily win and visualize the first exchange before each session.

### Strength & Power

**Phase:** GPP
**Weekly Progression:** Add 1 set or ~5–10% load weekly; deload final week by ~20%.
**If Time Short:** Keep top 2 lifts + 1 trunk/neck drill.

**Top Exercises:** Farmer’s Carry; Suitcase Carry Holds; Half-Kneeling Anti-Rotation Hold; 🔥 Single-Leg Rotational Med Ball Hold; Single-Arm Chest Press (Floor); Farmer Walk Hold (Static); Single-Leg Banded Anti-Extension Hold; Russian Twist (Weighted); Banded Lateral Walk; DB Alternating Incline Press (30-45°); DB Split Squat; Neck Harness Isometrics

**Prescriptions by Exercise Type:**
- **Ballistics (Med Ball / Speed / Power):** 3–5x4–6 reps (or 6–10 throws) at crisp intent; rest 60–90s.
- **Isometrics:** 3–5 holds x 10–20s @ 7–9/10 effort; full rest between holds.
- **Core Control:** 2-4 sets x 6-10 reps or 20-40s tempo (3-1-3), RPE 6-8.
- **General Strength:** 2–3x6–10 @ RPE 6–7, keep reps crisp.

**Adjustment:** ⚠️ Moderate fatigue → reduce 1 set if performance drops.

### Conditioning

#### Conditioning Block GPP — Aerobic Base + Glycolytic Power + Alactic Speed + Repeatability
**Intent:** Build aerobic base, improve repeatability, low damage.
**Dosage Template:** 3–5 rounds of 3–5 min @ RPE 6–7, work:rest 1:1–1:0.5 (cap 20–30 min). Template applies unless a drill lists its own structure.
**Weekly Progression:** Add 1 round or ~5–10% volume weekly; deload final week by ~20%.
**If Time Short:** Keep 2 aerobic rounds + 1 alactic pop.
**If Fatigue High:** If fatigue high: drop 1–2 rounds, keep intensity easy.

## PHASE 2: SPECIFIC PREPARATION PHASE (SPP) - 3 WEEKS (21 DAYS)

### Mindset Focus

- **Confidence:** Simulate competition pace and finish with a confidence recap of best rounds.

### Strength & Power

**Phase:** SPP
**Weekly Progression:** Hold volume, increase intensity or bar speed weekly; deload final week by ~20%.
**If Time Short:** Keep heavy lift + paired explosive + trunk.

**Top Exercises:** Farmer Carry (Heavy); Overhead Slam (Rotational); Clinch Off-balance + Knee Entry Patterning (Band-Resisted); Med-Ball Rotational Slam; Banded Row (Speed Focus); DB Clean & Press

**Prescriptions by Exercise Type:**
- **Ballistics (Med Ball / Speed / Power):** 4–6x2–5 reps at max speed; full rest 60–120s.
- **Core Control:** 2-4 sets x 6-10 reps or 20-40s tempo (3-1-3), RPE 6-8.

**Adjustment:** ⚠️ Moderate fatigue → reduce 1 set if performance drops.

### Conditioning

#### Conditioning Block SPP — Aerobic Base + Glycolytic Power + Alactic Speed + Fight-Pace Density
**Intent:** Match fight demands with intervals and skill-relevant density.
**Dosage Template:** 4–6 rounds of 2–5 min @ RPE 7–8, work:rest 1:1–1:0.5 (cap 18–25 min). Template applies unless a drill lists its own structure.
**Weekly Progression:** Increase density or intensity weekly; keep volume flat; deload final week by ~20%.
**If Time Short:** Keep 2 fight-pace rounds (system priority).
**If Fatigue High:** If fatigue high: drop volume, keep rest longer.

**System: AEROBIC** (scaled by format emphasis)
- **Drill: Rope Clinch Frames (heavy_bag)**  • Load: low (Bag must touch ropes/wall)<br>  • Rest: —<br>  • Timing: 30s rope lean → 30s active framing → x8 rounds<br>  • Purpose: Simulates late-round clinch against ropes (RPE 5)<br>  • Red Flags: None

**System: GLYCOLYTIC** (scaled by format emphasis)
- **Drill: Kill Mode Knees (bands)**  • Load: max (Bands anchored behind at ankle height)<br>  • Rest: —<br>  • Timing: 20 band-resisted knees → 10 switch knees → x5 rounds<br>  • Purpose: Bands increase hip flexor endurance for all combat sports knees. (90% effort)<br>  • Red Flags: None

**System: ALACTIC** (scaled by format emphasis)
- **Drill: Explosive Boxing Burst Intervals**  • Load: RPE 8–9, keep quality high and stop before speed drop-off<br>  • Rest: 75-120 sec complete rest between reps<br>  • Timing: 6–8 x 6–10 sec fast punch bursts<br>  • Purpose: Minimal-dose ATP-PCr maintenance for striking speed and neural sharpness.<br>  • Red Flags: Terminate set if punch speed/position quality drops.

**If Time Short:** Keep 2 fight-pace rounds (system priority).
**If Fatigue High:** If fatigue high: drop volume, keep rest longer.

**If Time Short:** Keep 4–6 alactic bursts + shadowboxing rhythm.
